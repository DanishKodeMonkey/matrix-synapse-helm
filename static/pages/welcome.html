Welcome to Danish Kode Matrix

This page covers first-login setup and where to go next.

1. Sign in

  • Open Element at https://element.danishkodemonkey.net.
  • Use homeserver matrix.danishkodemonkey.net if asked.
  • Complete registration using your invite/token workflow.

2. Secure your account

  • Set recovery and cross-signing as soon as possible.
  • Use a unique password and enable any available device security.
  • Review active sessions and remove unknown devices.

3. Join core rooms

  • #announcements:matrix.danishkodemonkey.net
  • #rules:matrix.danishkodemonkey.net
